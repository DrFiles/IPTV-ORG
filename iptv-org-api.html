How to Use Your API Key

Your API key allows you to access a variety of TV channels available through the IPTV Org API. To start using the API, follow these steps:

  1. Include your API key in the request header as follows:
    2. Authorization: Bearer YOUR_API_KEY
  2. Make a GET request to the following endpoint to retrieve channel information:
    3. GET https://api.iptv-org.com/channels
  3. The response will include a list of TV channels that you can integrate into your application or media player.

For more detailed documentation, visit the API Documentation page.
